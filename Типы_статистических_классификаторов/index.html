Типы статистических классификаторов

На форме Типы статистических групп можно создавать неограниченное количество типов классификаторов.

Для этого, необходимо нажать кнопку Добавить ()

Заполнить поля в открывшейся форме.

После чего, нажать кнопку ОК.

Таким образом, вы создаете список статистических классификаторов.
