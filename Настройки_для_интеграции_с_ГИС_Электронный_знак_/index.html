Настройки для интеграции с ГИС "Электронный знак"

В Администрирование – Интеграция вкладка ЭЗ необходимо указать данные доступа к личному кабинету ГИС МТ и авторизоваться.

Url к Web-сервису ГИС "Электронный знак" – адрес api доступа к ГИС ЭЗ

Пользователь – логин пользователя (адрес электронной почты)

Пароль для API – пароль для входа в личный кабинет

Host - IP адрес подключения к web-сервису

Токен – заполняется автоматически при успешной авторизации

Тестовый Web-сервис – указываем признак, если в Url и Host прописывали адрес к тестовой среде ЭЗ.

Лимит количества заказываемых марок по строке – можно установить максимальное количество КМ для товарной единицы в заказе.

Авторизация в ЭЗ – кнопка авторизации. После заполнения данных доступа необходимо нажать кнопку для авторизации в личном кабинете. После успешной авторизации будет заполнен токен.

Использовать организации - если у вас в системе несколько собственных организаций (компаний), и каждая работает с маркированным товаром, то настройки необходимо прописать для каждой организации. Когда отмечен признак Использовать организации отображается таблица со всеми компаниями, где для каждой компании можно настроить интеграцию, указав данные доступа в соответствующей ячейке.

Примечание

Повторная авторизация понадобится в случае, если вход в личный кабинет ГИС "Электронный знак" будет выполнен не из интерфейса lsFusion ERP

В карточке групп товаров, для которых используется маркировка, надо выбрать Тип маркировки.
