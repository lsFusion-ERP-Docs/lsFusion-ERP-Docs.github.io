Расчет рекомендации на закупку

В программе заложен базовый механизм расчета рекомендаций на закупку, который основан на анализе продаж, необходимых страховых запасах, имеющемся остатке и сроков следующей поставки. Данный расчет используется, как при формировании заказов вручную, так и при создании автозаказов (с использованием планировщика). Ряд промежуточных параметров, участвующих в расчете рекомендации можно увидеть при создании заказа, на вкладке Подбор - Товары.


Рис. 1 Параметры расчета рекомендации на закупку на вкладке Подбор

Ряд входных параметров задается на форме Закупки - Закупки - Параметры автозаказа. Параметры могут быть заданы для любого узла основного классификатора товаров.


Рис. 2 Форма Параметры автозаказа

  • Количество дней для расчета однодневных продаж - период (с текущей даты по текущая дата - указанное количество дней), за который анализируется реализация товаров. Если этот период не задан, то расчет среднедневных продаж не будет выполнен.
  • Количество дней для расчета сезонного коэффициента продаж - периоды до и после текущей даты прошлого года.
  • Коэффициент сезонности (упр.) - коэффициент, изменяющий количество к заказу.
  • Z-оценка (упр.) - коэффициент, определяющий вероятностный уровень доступности товара: 99% - 2,575; 95% - 1,95; 90% - 1,645; 70% - 1,035; 50% - 0,675.
  • Акционный коэффициент - коэффициент, отражающий предполагаемое влияние акции на темпы продаж.
  • Минимальное количество дней после акции - если между акциями малый промежуток (0-1 день, или <=указанному в параметре), то система считает их одной акцией. По умолчанию (если не заполнен на форме) значение параметра =1.

Параметры Дата поставки и Дата следующей поставки хранятся в заказах, на вкладке Дополнительные параметры. При формировании автозаказа эти даты берутся из Графика поставок (закупка).

Рассчитанное рекомендуемое количество к заказу будет округляться до величины кратной упаковке (поле Количество в упаковке (закупка) в карточке товара). Если в карточке товара заполнено поле Минимальное кол-во заказа, то рекомендуемое количество кратно значению в этом поле.
