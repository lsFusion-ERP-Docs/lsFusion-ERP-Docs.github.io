Заказ кодов маркировки

Для получения КМ на товары из ГИС ЭЗ необходимо сформировать и отправить заказ на марки. Заказ считается выполненным после получения списка КМ.

1. Создайте заказ на коды маркировки​

В модуле Склад на форме Заказы Марок с помощью кнопки Добавить создайте новый заказ на марки

Заполните шапку и строки заказа. Сохраните заполненный заказ.

Вид средства идентификации - выберите Код РБ идентификации товара (обувь, шины, промышленность) для получения марок РБ или Код стран ЕАЭС идентификации товара для получения марок РФ.
Тип - выбор типа запрашиваемых КМ из списка доступных.
Склад - выбор склада, на который запрашиваются КМ.
Кол-во - Количество единиц товара, для которых запрашиваются КМ.
GTIN – для каждого товара должен быть указан GTIN, товар должен быт описан в «Банке электронных паспортов товаров» – ePASS, иначе при отправке заказа в ГИС ЭЗ будет возникать ошибка.
Заполнить из накладной – по кнопке спецификацию можно заполнить определенными строками из выбранной накладной. Откроется окно выбора позиций накладных (закупка), где в поле Номер необходимо ввести номер накладной, позиции из которой надо добавить в заказ на марки. По умолчанию включен фильтр Позиции с заданной декларацией и отображаются только те строки, для которых есть данные по таможенной декларации. Чтобы добавить строки в заказ, их необходимо отметить в колонке Вкл. Чтобы выбрать строки без декларации, необходимо снять фильтр Позиции с заданной декларацией и отобразятся все строки выбранной накладной.
Данные декларации отображаются в соответствующих колонках в спецификации заказа марок.

2. Отправьте заказ в ГИС ЭЗ​

Для сохраненного заказа на форме Заказы марок, будет доступна вкладка Электронный знак. Необходимо перейти на вкладку и нажать кнопку Отправить в ЭЗ. При успешном выполнении запроса в ГИС ЭЗ будут созданы заказы на КМ по каждому GTIN на указанное в заказе количество. Номера созданных заказов будут отображены в колонке Код заказа ЭЗ.

Примечание

Если ранее не было заказов на КМ для товаров с GTIN в заказе, может отобразиться сообщение об ошибке, что такой товар не найден, а следующей строкой сообщение о добавлении такого товара в кабинет и, соответственно, заказ.

Если возникает следующая ошибка, то необходимо перечисленные GTIN проверить, зарегистрированы ли они в ePASS. Если не зарегистрирован хотя бы один GTIN, то весь заказ отклоняется. В таком случае, лучше разделить заказы - отдельно позиции по которым ждут регистрацию в epass, и отдельно по которым заказ марок пройдет.

После успешного обмена с ГИС ЭЗ на форме Заказы марок вкладке Электронный знак будет доступна кнопка Проверить статус. Необходимо выполнить проверку статуса. В результате проверки будет заполнен Статус по заказу каждого GTIN и указано количество Доступных марок. А на вкладке Электронный знак будет доступно действие Получить марки.

3. Получите КМ​

После перехода заказа в статус Выполнен на вкладке Электронный знак будет доступно действие Получить марки.

По действию Получить марки из Гис ЭЗ будет получен список КМ и в спецификации заказа отобразится количество полученных КЗ, а заказ будет автоматически переведен в статус Проведен.

Полученные КМ и их статус отображается на форме Склад - Марки.
