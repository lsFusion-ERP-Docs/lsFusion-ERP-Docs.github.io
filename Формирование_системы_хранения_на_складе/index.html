Формирование системы хранения на складе

Чтобы использовать адресное хранение необходимо задать структуру ячеек адресного хранения, которая соответствует складу. Для этого предназначены формы WMS - Группы ячеек и WMS - Ячейки.

Ячейка - минимальный элемент структуры. Ячейки объединяются в группы, которые могут соответствовать складским зонам, стеллажам и т.д.

Группы ячеек​

Группы ячеек создают на форме WMS - Группы ячеек.

Чтобы создать группу ячеек используют кнопку Добавить. Для группы ячеек указывают:

Наименование - удобное название группы ячеек

Код - код элемента.

Родительский объект - группа ячеек на 1 ступень иерархии выше, в которую включена текущая группа ячеек

Непосредственно ячейки хранения создаются на форме WMS - Ячейки.

Создание и настройка ячеек​

На форме WMS - Ячейки отображается структура складов и ячейки, которые находятся в выделенном элементе структуре. Чтобы Добавить ячейки в группу ячеек в блоке Ячейки нажмите Добавить и заполните данные :

Наименование - удобное название группы ячеек

Код - код ячейки

Группа - группа ячеек, в которой находится текущая ячейка

Для каждого склада/отдела магазина, где используется адресное хранение, обязательно нужно отметить группы ячеек, которые там используются.

Также, чтобы автоматически приходовать товар в предназначенное для него место и упростить процесс комплектации и отгрузки, за товарами или группами товаров можно закрепить определенные ячейки, которые будут автоматически подтягиваться в документы прихода и отгрузки.

Для настройки необходимо:

  1. в поле Наименование указать склад, для которого задаются настройки
  2. отметить в блоке Группы ячеек те группы, которые находятся на выбранном складе. если отмечается группа ячеек, то автоматически отмечаются все ячейки. Можно также отметить отдельные ячейки, а не всею группу, в блоке Ячейки
  3. все внесенные на форму изменения необходимо Сохранить
