Создание прайса

Для создания прайса необходимоЖ

  1. Зайти на форму Цены - Документы - Прайсы
  2. Нажать кнопку Добавить
  3. В открывшейся форме заполнить поля: Операция, Период действия, Дополнительные параметры
    • В поле Операция после клика мышью выбираем необходимую нам операцию:
    • В поле Период действия вносим Дата с и Дата по
    • В области Дополнительные параметры мы можем настраивать дополнительные свойства, действующие для каждой строки прайса
Примечание

Для каждой операции настраивается своя комбинация дополнительных параметров.

  1. Выбор склада, для которого будет действовать прайс, осуществляется во вкладке Склады
  2. Виды цен, которые будут учитываться в прайсе, отмечаются во вкладке Вид цен
  3. Включение товаров в прайс производится на вкладке Подбор, путем внесения отметки в поле Вкл
  4. Импорт прайсового списка товаров может производится на вкладке Импорт

После включения необходимых товаров в прайс, а так же внесения всех требуемых настроек, необходимо сохранить прайс и нажать Провести для вступления его в силу.
