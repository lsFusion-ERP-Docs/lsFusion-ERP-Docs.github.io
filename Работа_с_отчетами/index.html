Работа с отчетами

В работе с отчетными формами действует один и тот же принцип :

Заходим на форму и выполняем следующую последовательность :

Выбираем Дату либо Период, за который необходимо сформировать отчет

Выбираем Объект, по которому формируем отчет (Склад или Магазин)

Выбираем Группу товаров либо Все товары.

Нажимаем кнопку формирования Печатной формы

Далее рассмотрим работу более подробно.

Товарный отчет​

Заходим на форму «Товарный отчет» модуль Склад раздел Отчеты.

Выбираем Диапазон дат. Выбираем Склад (Отдел). Нажимаем кнопку «Распечатать».

В отдельном окне сформируется печатная форма отчета. Далее нам необходимо нажать на кнопку «Принтер» и выбрать печатающее устройство.

Примечание 1
На форме имеется возможность распечатать товарный отчет в разрезе операций по кнопке «По операциям», а так же отчет по таре.

Отчет по таре​

Заходим на форму «Отчет по таре» модуль Склад раздел Отчеты.

Выбираем Диапазон дат, Выбираем Склад (Отдел магазина). Нажимаем «Распечатать».

Отчет по товарообороту​

Заходим на форму «Отчет по товарообороту» модуль Склад раздел Отчеты.

Выбираем Диапазон дат. Выбираем Склад. В дереве Классификатора отмечаем Группу товаров или все Товары.

Нажимаем кнопку печати формы «Списком» или «По группам».

Отчет по остаткам​

Заходим на форму «Отчет по остаткам» модуль Склад раздел Отчеты.

Выбираем Дату. Выбираем Склад. Отмечаем Группу товаров. Нажимаем кнопку «Списком» или «По группам».

Отчет по поставщикам​

Заходим на форму «По поставщикам» модуль Склад раздел Отчеты.

Отбираем Диапазон дат. Выбираем Поставщика. Отмечаем Склад. Отмечаем Группу товаров или Все товары. Нажимаем кнопку «Отчет по поставщику».

Отчет по поступлениям​

Заходим на форму «Отчет по поступлениям» модуль Закупки раздел Отчеты.

Выбираем Период. Выбираем Склад. Отмечаем Группу товаров. Переходим в закладку Печать. Нажимаем нужную кнопку для формирования отчета.

Отчет по продажам​

Заходим на форму Отчет по продажам модуль Продажи раздел Отчеты.

Выбираем Период. Выбираем Склад. Отмечаем Группу товаров. Переходим в закладку Печать. Нажимаем нужную кнопку для формирования отчета.

Отчет по магазинам​

Заходим на форму Отчет по магазинам модуль Розничная торговля раздел Отчеты.

Выбираем Период. Отмечаем Склад (Отдел магазина). Нажимаем кнопку Печати таблицы либо Экспорта в Эксель.
