Товарный отчет

Товарный отчет формируется на форме *Склад - Товарный отчет *

Для получения информации, необходимо ввести интервал дат и выбрать отдел, по которому будет сформирован отчет.

Также, на форме есть кнопка Закрыть документы до Дата(по) и поле для внесения даты. Эта функция позволяет ответственному лицу после выверки данных, присвоить документам статус Закрыт до указанной даты+1.

То есть в данном примере дата по стоит 09.06, следовательно по нажатию кнопки Закрыть документы до Дата (по), статус "закрыт" будет присвоен документам до 10.06 включительно.

Присвоенный статус "закрыт" позволяет избежать корректировки документов после выверки данных.

Подробнее читать о закрытии документов/ периода.
