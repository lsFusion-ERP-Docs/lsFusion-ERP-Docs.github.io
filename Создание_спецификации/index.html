Создание спецификации

Чтобы создать новую спецификацию необходимо в списке спецификаций нажать кнопку Добавить.

Откроется форма спецификации в которой необходимо заполнить данные.

Шапка документа​

Наименование - введите наименование спецификации, чтобы ее можно было легко идентифицировать среди других. Обычно наименование спецификации связано с названием производимого изделия.

Дата с/по - период действия спецификации.

Операция - операцию спецификации обязательно необходимо указывать. Кликните в поле и выберите подходящую из доступных операций. Операции настраиваются на форме Производство - Операции.

Отображать энергетическую ценность - если включено это свойство, на вкладке Энергетическая ценность/Влажность отображается состав БЖУ и калорийность компонентов и рассчитывается энергетическая ценность конечного блюда

Номер по сборнику рецептов - здесь можно указать номер рецептуры в соответствии с вашим справочником.

СТБ - выберите стандарт, по которому составлена спецификация.

Отображать влажность - если включено это свойство, на вкладке Энергетическая ценность/Влажность отображается влажность компонентов и готовой продукции

Номер, Серия, Дата и Время документа будут заполнены автоматически.

Примечание - в поле можно внести любые дополнительные данные

Компания - в поле можно указать компанию, для которой действует данная спецификация.

Вкладка Изделие​

На вкладке Изделие необходимо указать товар, который будет произведен по спецификации. Для этого необходимо нажать кнопку Добавить, появится пустая строка. кликните в поле SKU и выберите товар из списка.

Далее необходимо указать Количество, которое будет произведено по спецификации. Вес будет рассчитан исходя из данных в карточке товара.

Укажите Удельный вес или Коэффициент цены, неуказанный показатель будет рассчитан автоматически. Данные показатели определяют какая часть цены компонентов будет учтена в цене готового изделия.

Вкладка Компонент​

На вкладке Компонент необходимо указать сырье для производства изделия.

Для каждого компонента надо указать необходимое количество (Кол-во (нетто) для производства готового изделия в количестве, указанном на вкладке Изделие.

Если компонент расходуется не весь, а какая-то его часть утилизируется, то необходимо указать процент потерь. Например, картофель используется очищенным, очистки составляют 10% от общей массы картофеля. Такие потери можно задавать непосредственно на вкладке, проставив значение в колонку % потерь, либо указать используемую Технологию . В обоих случаях программа рассчитает необходимый вес и количество брутто.

Для добавления компонентов можно использовать кнопку Добавить и вводить товары по-одному, или перейти на закладку Побор и проставить количество нетто для используемых компонентов. Они отобразятся в списке компонентов.

В качестве компонентов могут использоваться также готовые изделия.

Внимание

Обратите внимание, что если в процессе производства из целого получают несколько товаров (например, при разрубе туши), то на вкладке Изделие будут указаны те товары, которые получают и процент цены товара-компонента, который распределяется на каждое изделие, а на вкладке Компонент будет указан товар, из которого получают изделие.

Вкладка Склады​

На этой вкладке можно ограничить склады, для которых будет действовать данная спецификация. Она будет доступна только тем складам, которые отмечены. Отметить можно сразу группу складов в Дереве, или отдельные вклады в блоке Склад.

Вкладка Акт контрольной проработки​

На данной вкладке необходимо внести данные для печати акта контрольной проработки.

Вкладка Энерг.ценность/Влажность​

Данная вкладка становится доступной если в спецификации отмечено свойство Отображать энергетическую ценность или Отображать влажность . Программа рассчитывает и отображает энергетическую ценность изделия на основе данных по БЖУ и калорийности, которые указаны в карточках товаров-компонентов. Влажность рассчитывается на основании указанной на вкладке влажности компонентов.

Чтобы рассчитанные данные по энергетической ценности отобразились в карточке товара-изделия, нажмите кнопку Записать в карточку товара.

Вкладка Описание​

На данной вкладке отображается описание состава товара-изделия. Нажмите кнопку Заполнить описание и в поле Описание будет добавлен список компонентов.

Описание состава можно перенести в карточку товара-изделия, для этого служит кнопка Записать описание в карточку товара. Оно отобразится в поле Описание и его можно будет напечатать на этикетку или ценник.

Вкладка Технологическая инструкция​

На данной вкладке можно составить и хранить технологическую инструкцию по изготовлению изделия. Составленную инструкцию можно распечатать на форме Спецификации. Для каждого раздела инструкции предусмотрена отдельная закладка.

Вкладка Протокол​

Данная вкладка служит для ввода данных для печати протокола дегустационной комиссии.

Примечание

Составляя спецификацию периодически сохраняйте проделанную работу, это позволит избежать случайной потери данных.
