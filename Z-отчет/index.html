Z-отчет

Находится в Розничная торговля - Z-отчеты

Данная форма позволяет получить информацию

  • о кассовой ленте с детализацией по строкам чека без ограничения периода
  • об оплатах по чекам различных типов (наличные, банковская карточка)
  • о суммах предоставленных скидок по чекам
  • о продажах в разрезе кассовых секций (секции присваиваются товару в разделе Розничная торговля → Магазины )

Описание​

Форма разделена на 5 логических блоков:

  1. Фильтры - в этой области размещены поля ввода данных для отбора значений свойств по заданному условию.
  2. Z-отчет - перечень Z-отчетов со значениями статусов и свойств.
  3. Чек - содержит в себе перечень всех чеков со значениями свойств, принадлежащих каждому отдельному Z-отчету
  4. Строка чека - по позиционно расписаны проданные товары, принадлежащие одному чеку
  5. Оплата по чеку - указывается тип платежа и форма оплаты.

Структура​

Фильтры - основные фильтры:

  • Дата с , Дата по - задание диапазона дат для отбора необходимых Z-отчетов

  • Отдел магазина - отбор Z-отчетов по отделам

  • Касса Z-отчета - отбор по номеру кассы

    Z-отчет (открытый) - содержит информацию о Z-отчетах

  • Проверено - статус прохождения проверки

  • Закрыт - статус открытости\закрытости Z-отчета

  • Проведен - статус проведения в системе

  • Номер - номер Z-отчета

  • Дата документа - Дата Z-отчета

  • Время документа - Время Z-отчета

  • Отдел магазина - отдел магазина

  • Касса Z-отчета - номер кассового аппарата

  • Основание - информация о Z-отчете

  • Количество строк - количество записей в структуре Z-отчета по чекам

  • Количество (всего) - количество записей в структуре Z-отчета по строкам чеков

  • Сумма Z-отчета - общая сумма по Z-отчета

  • Сумма продаж (наличные) - сумма продаж за наличные

  • Сумма продаж (карточкой) - сумма продаж за безнал.

  • Сумма возврата (наличные) - сумма возврата наличных средств

  • Сумма возврата (карточкой) - сумма возврата безналичных средств

  • Сумма НДС - сумма НДС

  • Сумма скидок Z-отчета - сумма скидок по Z-отчету

В нижней части информационного контейнера Z-отчет (открытый) имеется один фильтр и несколько функциональных кнопок:

Фильтр Открыт - во включенном состоянии (посредством проставления галочки) на форме будут отображены только Z-отчеты, не имеющие статуса "Закрыт"

Кнопки:

  • Закрыть - переводит Z-отчет в закрытый статус
  • Распровести - отменяет проведение операций по данному Z-отчету (если Z-отчет закрыт, то предварительно необходимо перевести его в открытый статус)
  • Добавить - добавляет новый Z-отчет
  • Редактировать - редактирует текущий Z-отчет (осуществляется переход на новую форму редактирования Z-отчет (открытый))
  • Экспортировать чеки - экспортирует чеки во внешний файл
  • Импортировать чеки - импортирует чеки из внешнего файла определенного формата и структуры

Чек - отображает информацию о количестве чеков, имеющихся в Z-отчете и их содержание.

  • Номер чека
  • Дата документа
  • Время документа
  • Кассир
  • Номер дисконтной карты
  • Держатель дисконтной карты
  • Сумма чека
  • Сумма скидки
  • Сумма скидки по чеку
  • Сумма НДС
  • Кол-во строк
  • Кол-во всего
  • Сумма продажа (наличные)
  • Сумма продажа (карточка)

Строка чека - отображает информацию о количестве позиций и их содержании в составе одного чека.

  • Тип
  • Штрихкод
  • Товар
  • Чек продажи
  • Количество
  • Возвращено
  • Цена
  • Сумма поставщика без НДС
  • Сумма
  • Процент скидки
  • Строка заказа

Оплата по чеку - отображает информацию о типе платежа и форме оплаты

  • Тип платежа
  • Форма оплаты
  • Сумма платежа
  • Расписано для строки

Условия акции - отображает условия проводимой акции для данного товара

  • Акция
  • Наименование
  • Действует (от)
  • Действует (до)
  • Процент скидки
  • Кол-во по акции
  • Сумма скидки
  • Применить скидку
