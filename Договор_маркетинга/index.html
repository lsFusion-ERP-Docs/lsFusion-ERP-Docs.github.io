Договор маркетинга

Договоры маркетинга отображаются на форме Рабочий стол - Маркетинг - Договоры маркетинга. Для создания и корректировки договоров предназначены кнопки Добавить, Редактировать, Удалить, Копировать. Также договоры можно импортировать.

По каждому договору в нижней части формы отображаются условия расчета маркетинговых платежей, на которые распределяются платежи.


Рис. 1 список маркетинговых договоров

В форме договора заполняются данные в шапке и условия расчета маркетинговых платежей в сопроводительных листах.

Заполнение шапки договора​

Рис. 2 Шапка договора маркетинга

Номер и серия договора могут быть внесены вручную или заполнятся автоматически при настроенном нумераторе.

Действует с/по - период действия договора, если договор бессрочный, поле Действует (по) можно не задавать. Начало действия договора нужно задавать всегда.

Наименование - используется для удобной идентификации договора

Пролонгация - признак пролонгации, используется при печати акта на оплату по договору

Периодичность выставления - периодичность формирования актов выполненных работ по договору: выбирается значение Месяц или Квартал.

Поставщик - поставщик товара, выступающий стороной договора, которой оказываются маркетинговые услуги.

Получатель - получатель товара, оказывающий маркетинговые услуги

Тип договора - выбирается одно из значений: Аудио-реклама, Информационные услуги, Рекламные услуги или Торговая премия. Тип договора определяют печатную форму для автоматически формируемого акта на оплату

Условия оплаты, От объема/От оплат/Обратный, В лице, действующего на основании - атрибуты, используемые в печатной форме акта

Валюта взаиморасчетов - необходимо указать валюту договора из списка.

Тип обмена - если валюта договора отличается от базовой, необходимо задать тип обмена валюты.

Загрузить/Просмотреть/Удалить - кнопки для загрузки, просмотра и удаления файла (docx, PDF или иного формата) оригинала договора.

Заполнение сопроводительных листов​

С помощью сопроводительных листов задаются условия расчета маркетинговых платежей для договора.

Для одного договора может быть задано несколько сопроводительных листов (например, для годового договора задаются ежемесячные сопроводительные листы в случае их пересмотра с такой периодичностью).

Для создания сопроводительного листа используется кнопка Добавить в нижнем блоке формы договора.


Рис. 3 Список сопроводительных листов договора

На форме Сопроводительный лист задается его номер и срок действия.

Для используемых видов маркетинга задаются условия на соответствующих вкладках Маркетинг розничный, Логистический бонус, Мерчендайзинг, Маркетинг фиксированный, Браки, Диапазон.


Рис. 4 Заполнение сопроводительного листа

Для Маркетинг розничный, Логистический бонус, Мерчендайзинг, Браки задается база расчета, необходимо выбрать из доступных вариантов

Рис. 5 База расчета

Далее добавляются строки, в которых в соответствующих графах задаются фильтры, с помощью которых формируются условия расчета маркетингового платежа и процент маркетинга (рис. 4). Фильтры применения текущего маркетингового условия действуют по правилу «И»: условие применится для конкретного товара при автоматическом формировании акта только в том случае, если все введенные фильтры будут выполняться.

Можно задавать условия-исключения, например для всей группы товаров действует одно условие и процент маркетинга, а для определенного склада - другое. В таком случае необходимо отметить признак Исключение и указать условие.

Для одного сопроводительного листа можно одновременно задавать условия маркетинга разных типов (например, одновременно Маркетинг розничный, Логистический бонус и Маркетинг фиксированный). При этом для каждого типа маркетинга может быть введено несколько разных условий.

Часть фильтров относится к товарам: Категория, Бренд, Страна изготовитель, Вес, Товар

Другая часть к подразделениям: Регион, Юр. лица, Группы складов, Склады.
