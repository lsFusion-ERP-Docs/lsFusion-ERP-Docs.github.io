Группы касс

В группу касс объединяются кассовые аппараты одного типа (модель касс) на одном торговом объекте. Для группы касс настраиваются общие параметры и прописываются входящие в нее КСА. Группы касс отображаются на форме Оборудование - Кассы - Группы касс.


Рис. 1 Группы касс

В группе касс устанавливаются следующие параметры (рис. 3) :

  • Порядковый номер – сквозной, уникальный код для всех групп оборудования.

  • Номер группы касс – вспомогательный код, используемый для некоторых типов касс (УКМ).

  • Склад – отдел магазина, где используется данная группа оборудования.

  • Наименование – наименование группы касс.

  • Код для штучных товаров, Код для весовых товаров – префикс для штрихкода на штучный/весовой товар.

  • Разрешить прием реализации по закрытым документам – отметка актуальна, если для Сервера оборудования стоит отметка Не принимать реализацию по закрытым документам

  • Фильтровать по классификатору – при отметке появляется кнопка «Выбрать группы» – позволяет ограничить список товарных групп, загружаемый в оборудование.


    Рис. 2 Выбор групп для фильтрации по классификатору

  • Вид цен – вид цены, которая передается в кассы.

  • Модель – модель кассы, к которой относится оборудование.

  • Директория обмена с кассами – формат ввода зависит от модели кассы.

  • Дата начала продаж – начиная с указанной даты, принимаемая реализация будет влиять на изменение остатка. Z-отчеты сохраняются без учета данной даты.

  • Неактивный – если признак отмечен, то по группе касс не ведется обмен данными с сервером.

  • Приоритет – порядок обработки групп касс при засылке цен. Чем больше число, тем раньше засылка. Для некоторых моделей касс (Artix) поддержан приоритет и по приему реализации.

  • Использовать в POS вид цены напрямую – нужно отметить, если используется встроенный POS (модуль "Управление продажами"). В этом случае данные о цене доступны напрямую из системы. Если отметка не стоит, то используются последние цены из формы Загрузка прайса в оборудование.

  • Разрешить повторную продажу сертификата - если используется встроенный POS (модуль "Управление продажами"), то при отмеченном признаке один и тот же сертификат может быть продан больше одного раза.

  • Отдел продажи сертификата - используется в настройке для касс Sento и TitanPOS.

  • Выгружать признак маркировки - если отмечен признак, то в кассовый сервер выгружается признак, что товар подлежит маркировке.

  • Сервер оборудования – внешняя служба, поддерживающая обмен с определенным типом торгового оборудования.


Рис. 3. Настройка группы касс

Вкладка Касса​

На вкладке перечисляются все устройства, относящиеся к данной группе. Нужно создать новую строку кнопкой Добавить и обязательно указать группу и порядковый номер устройства. Рекомендуем также заполнять описание устройства, чтобы быстро и легко ориентироваться в оборудовании. Далее, в зависимости от устройства, устанавливаются остальные параметры, например, пути для логирования, порты подключения, коды платежей, платежные терминалы и т.д.

Рис. 4 Вкладка Касса

При использовании фискального регистратора (ФР), в поле «Модель» указывается соответствующий тип регистратора и ставится отметка «Использовать в POS вид цены напрямую». Данная отметка позволяет использовать все товары в БД LSFUSION, имеющие цену заданного вида. Если отметка не стоит, то используются последние цены из формы «Загрузка прайса в оборудование».

В блоке Обычный пользователь отображаются пользователи системы и можно задать номер кассира для некоторых моделей фискальных регистраторов, внеся значение в поле Номер в группу касс.


Рис. 5 Блок Обычный пользователь

Настройка POS на рабочей станции​

Для того, чтобы с компьютера (кассового устройства) была возможность работать на форме POS, необходимо на вкладке Касса для подключенного устройства указать в поле Компьютер сетевое имя ПК. Чтобы убедиться, что имя указано правильно, можно войти в lsFusion ERP через клиентское приложение, перейти на форму Администрирование - Доступ - Компьютеры и включить фильтр Тек. компьютер. Сетевое имя компьютера указано в поле Имя хоста.
